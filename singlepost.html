Exam Structure

How the Exam will be conducted?


The whole exam will contain 2 parts in which the first part will have objective type questions and the second part will contain the subjective type questions. The pattern of the exam and the question type is as mentioned below:

PART I

Each subject will have 15 questions which will be objective types.
Marks distribution (for easy questions):
Ø For each correct answer 1 mark will be awarded.
Ø For each wrong answer 0.25 marks will be deducted.

Marks distribution (for medium questions):
Ø For each correct answer 1 mark will be awarded.
Ø For each wrong answer 0.50 marks will be deducted.

PART -II

Each subject will contain 15 objective type question and all the question will be of hard level.

Marks distribution (for hard questions):
Ø For each correct answer 2 mark will be awarded.
Ø For each wrong answer 0 marks will be deducted.
